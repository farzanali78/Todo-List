# Project Description

This is a simple yet functional To-Do List App built using HTML, CSS, and JavaScript. While creating this project, I learned how to effectively use DOM manipulation to dynamically create, update, and remove elements in the browser.

The app allows users to quickly add, view, and delete tasks with an intuitive interface. All tasks are stored in localStorage, so your list persists even after refreshing the page.

##  Features

- Add new tasks dynamically

- Mark tasks as completed

- Delete tasks individually

- Persistent data storage using localStorage

- Responsive and clean UI built with CSS

- Real-time DOM updates without page reload

## What I Learned

How to select and manipulate DOM elements using querySelector and getElementById

How to dynamically create, append, and remove HTML elements with JavaScript

How to attach event listeners for interactivity (add, delete, mark complete)

How to use localStorage to save data across sessions

Basic styling and layout with CSS Flexbox
